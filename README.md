# [Project 2: Data Professional Survey Analysis Dashboard](https://drive.google.com/drive/u/0/folders/1lVFgUnr4SA64mDrRtqAeK2OAA4vebtyf)

### Overview

Designed and developed an interactive Power BI dashboard to analyze a global survey of data professionals. The project transformed raw survey data into actionable insights, enabling the exploration of salary trends, work-life balance, programming language preferences, career satisfaction, and barriers to entry within the data industry.

### Business Objective

Organizations, recruiters, and aspiring data professionals often seek to understand industry trends, compensation patterns, and workforce preferences. This project addresses these needs by providing a comprehensive visual analysis of survey responses from data professionals across multiple countries.

### Key Contributions
Cleaned and transformed survey data using Power Query.
Designed an interactive dashboard to communicate insights effectively.
Developed KPI visualizations to measure salary and work-life balance satisfaction.
Performed comparative analysis across job roles, countries, and programming language preferences.
Applied data storytelling techniques to highlight meaningful workforce trends.
Created a user-friendly dashboard layout focused on business decision-making.
Technologies Used

Power BI • Power Query • DAX • Data Cleaning • Data Visualization • Business Intelligence

### Key Insights
Data Scientists reported the highest average salaries among surveyed roles.
Python emerged as the most preferred programming language among data professionals.
The United States and India represented the largest share of survey participants.
Professionals reported higher satisfaction with work-life balance than salary.
Many respondents indicated that entering the data industry is moderately challenging.
Skills Demonstrated

Business Intelligence • Dashboard Development • KPI Analysis • Data Storytelling • Data Visualization • Power BI • DAX • Power Query • Insight Generation

### Impact

This project demonstrates the ability to transform raw survey data into executive-level insights through interactive dashboards, supporting data-driven decision-making and workforce trend analysis.
