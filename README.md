# [Project 2: Household Power Consumption Forecasting](https://github.com/GKA95/Household-Power-Consumption-Forecasting)


### Overview

Developed an end-to-end machine learning solution to forecast household electricity consumption using historical energy usage data. The project focused on identifying consumption patterns, engineering predictive features, and evaluating multiple forecasting models to improve the accuracy of residential energy demand predictions.

### Problem Statement

Accurate forecasting of household electricity demand is essential for energy planning, smart grid operations, and efficient resource allocation. Traditional forecasting methods often struggle to capture the complex and dynamic nature of residential energy consumption. This project addresses that challenge by leveraging machine learning techniques to predict future power usage based on historical consumption patterns.

### Key Contributions

* Collected and preprocessed household energy consumption data.
* Performed exploratory data analysis to uncover usage trends and seasonal patterns.
* Engineered time-based and consumption-related features to enhance predictive performance.
* Developed and compared multiple machine learning forecasting models.
* Evaluated model performance using industry-standard metrics.
* Generated actionable insights to support energy management and demand forecasting.

### Technologies Used

Python • Pandas • NumPy • Scikit-Learn • Matplotlib • Seaborn • Jupyter Notebook

### Results

* Identified significant daily, weekly, and seasonal consumption patterns.
* Improved forecasting performance through feature engineering and model optimization.
* Demonstrated the effectiveness of machine learning for residential load forecasting.
* Produced insights that can support smart grid planning and demand-side energy management.

### Skills Demonstrated

Machine Learning • Time-Series Forecasting • Data Analytics • Energy Analytics • Data Visualization • Feature Engineering • Statistical Analysis • Predictive Modeling

### Impact

This project demonstrates how data-driven forecasting techniques can improve energy planning, support smarter electricity distribution, and contribute to more efficient and sustainable power systems.

# [Project 3: Data Professional Survey Analysis Dashboard](https://drive.google.com/drive/u/0/folders/1lVFgUnr4SA64mDrRtqAeK2OAA4vebtyf)

### Overview

Designed and developed an interactive Power BI dashboard to analyze a global survey of data professionals. The project transformed raw survey data into actionable insights, enabling the exploration of salary trends, work-life balance, programming language preferences, career satisfaction, and barriers to entry within the data industry.

### Business Objective

Organizations, recruiters, and aspiring data professionals often seek to understand industry trends, compensation patterns, and workforce preferences. This project addresses these needs by providing a comprehensive visual analysis of survey responses from data professionals across multiple countries.

### Key Contributions
Cleaned and transformed survey data using Power Query.
Designed an interactive dashboard to communicate insights effectively.
Developed KPI visualizations to measure salary and work-life balance satisfaction.
Performed comparative analysis across job roles, countries, and programming language preferences.
Applied data storytelling techniques to highlight meaningful workforce trends.
Created a user-friendly dashboard layout focused on business decision-making.
Technologies Used

Power BI • Power Query • DAX • Data Cleaning • Data Visualization • Business Intelligence

### Key Insights
Data Scientists reported the highest average salaries among surveyed roles.
Python emerged as the most preferred programming language among data professionals.
The United States and India represented the largest share of survey participants.
Professionals reported higher satisfaction with work-life balance than salary.
Many respondents indicated that entering the data industry is moderately challenging.
Skills Demonstrated

Business Intelligence • Dashboard Development • KPI Analysis • Data Storytelling • Data Visualization • Power BI • DAX • Power Query • Insight Generation

### Impact

This project demonstrates the ability to transform raw survey data into executive-level insights through interactive dashboards, supporting data-driven decision-making and workforce trend analysis.
